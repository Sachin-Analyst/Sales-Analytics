# Sales-Analytics
This project involves a comprehensive sales performance analysis for AtliQ Hardwares, a global consumer electronics entity. The primary goal was to conduct a variance analysis between actual net sales and established targets to identify performance gaps and growth opportunities across diverse regions, markets, and divisions.
